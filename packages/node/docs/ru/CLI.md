# CLI

Пакет `@xmihome/node` включает в себя инструменты командной строки для
управления вашими устройствами Xiaomi.

## Команды

### `xmihome`

Основная команда для аутентификации и поиска устройств.

#### `login`

Позволяет в интерактивном режиме войти в Xiaomi Cloud. Успешная
аутентификация сохранит ваши учетные данные в локальный файл
(`~/.config/xmihome/credentials.json`) для будущего использования.

**Использование:**

```bash
xmihome login [options]
```

**Опции:**

- `-u, --username`: Имя пользователя вашего аккаунта Xiaomi.
- `-p, --password`: Пароль от вашего аккаунта Xiaomi.
- `-c, --country`: Код страны вашего аккаунта (например, `ru`, `us`, `cn`).

#### `devices`

Выводит список всех ваших устройств, обнаруживая их в локальной сети (miIO и
Bluetooth) и/или в Xiaomi Cloud.

**Использование:**

```bash
xmihome devices [options]
```

**Опции:**

- `--type`: Указывает тип обнаружения (`all`, `miio`, `bluetooth`, `cloud`).
- `--force`: Принудительно запускает поиск, игнорируя кэш.

### `xmihome-setup-bluetooth`

Утилита для упрощения настройки Bluetooth на системах Linux. Она генерирует
файлы политик D-Bus для предоставления разрешений и может настроить прокси
для удаленного доступа к Bluetooth.

**Использование:**

```bash
xmihome-setup-bluetooth [options]
```

**Опции:**

- `--remote`: Создать конфигурацию для удаленного доступа через TCP-прокси.
- `--port`: Порт для удаленного доступа (по умолчанию: 55555).
- `--host`: Хост для прослушивания (по умолчанию: 0.0.0.0).
- `--help, -h`: Показать справку.
